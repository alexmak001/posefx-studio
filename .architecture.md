# posefx-studio — Architecture & Implementation Status

## Pipeline Overview

```
webcam → [frame] → pose estimator  → PoseResult  → skeleton overlay  ─┐
                  → segmenter       → MaskResult  → mask overlay      ─┤→ composite frame → preview window
                                                     FPS counter      ─┘
```

## Module Map

### src/io/ — Input/Output
- **webcam.py** — `WebcamCapture`: OpenCV camera wrapper. Configurable device, resolution, FPS.
- **preview.py** — `PreviewWindow`: OpenCV display window with quit detection (q/ESC).
- **recorder.py** — *(stub)* Future video recording.

### src/inference/ — Model Inference
- **base.py** — Abstract interfaces: `BasePoseEstimator`, `BaseSegmenter`. Data contracts: `PoseResult`, `MaskResult`.
- **pose_estimator.py** — `YOLOPoseEstimator(BasePoseEstimator)`: YOLOv8-pose wrapper. Returns COCO 17-keypoint format.
- **segmenter.py** — `YOLOSegmenter(BaseSegmenter)`: YOLOv8-seg wrapper. Filters to person class (COCO class 0).

### src/tracking/ — Temporal Tracking
- **smoother.py** — *(stub)* Keypoint temporal smoothing.
- **pose_state.py** — *(stub)* Per-person pose state management.

### src/logic/ — Business Logic
- **gestures.py** — *(stub)* Gesture detection from pose data.
- **zones.py** — *(stub)* Spatial zone definitions and hit testing.
- **triggers.py** — *(stub)* Event triggers from gestures + zones.

### src/render/ — Rendering
- **debug_renderer.py** — *(stub)* Dedicated debug overlay renderer.
- **compositor.py** — *(stub)* Frame compositing pipeline.

### src/utils/ — Utilities
- **config.py** — `load_config()` → `AppConfig` dataclass tree. YAML-driven, typed access.
- **timing.py** — `FPSCounter`: Rolling average FPS over N frames.
- **geometry.py** — *(stub)* Geometry helpers.

### src/main.py — Entry Point
- Loads config, instantiates all pipeline stages, runs the frame loop.
- Draws: mask overlay → skeleton → FPS text.
- `python -m src.main --config config/demo.yaml`

## Data Flow Contracts

| Dataclass    | Fields                                          | Shape Info                    |
|--------------|-------------------------------------------------|-------------------------------|
| `PoseResult` | keypoints, confidences, boxes, num_people       | (N,17,2), (N,17), (N,4), int |
| `MaskResult` | masks, combined_mask, num_people                | (N,H,W), (H,W), int          |

## Config Structure (config/demo.yaml)

- `camera` — device_id, width, height, fps
- `inference` — pose_model, seg_model, device, confidence_threshold
- `debug` — show_skeleton, show_mask, show_fps, show_keypoint_confidence
